This website features an intelligent acoustic metamaterial design assistant based on large language models, currently including the following functions:

1. Reverse design of Acoustic Valley Hall Insulator (AVHI) unit cells based on user-provided topological bandgap ranges.

2. Answer user questions related to acoustics.

For detailed usage instructions, please refer to the Help documentation.
AVHI
Acoustic Valley Hall Insulator (AVHI) originates from the extension and evolution of topological band theory in condensed matter physics to acoustics, possessing topologically protected edge states. Under specific conditions, it can excite valley Hall effect, thus demonstrating efficient acoustic wave control capabilities. This characteristic gives acoustic valley Hall insulators important application potential in fields such as acoustic communication, acousto-electric integration, and sound source localization.
LLM Training Data Source
The LLM's supervised fine-tuning training data comes from a reinforcement learning-based Acoustic Valley Hall Insulator optimization design method. Compared to previous optimization algorithms, this method has the following two main advantages:

1. Reinforcement learning can obtain more diverse design results through exploratory design, which facilitates the construction of valley Hall insulator model libraries and provides users with greater freedom of choice.

2. Based on the model learning reasonable design experience, as the number of model training rounds continues to increase, theoretically it can continuously break through the current known design limits.
Currently this project is in the testing phase and not the official release version. The backend service is deployed on a small server and may experience access delays.

🎯 Project Introduction

XYQX is an AI Agent assistant based on large language models, currently specializing in Acoustic Valley Hall Insulator (AVHI) design, and will continue to expand to other acoustic metamaterials.

  1. Web development framework: Frontend web interface developed based on Python Flask

  3. Select function: Click buttons in the navigation bar to switch between different function pages
  4. Start designing: Enter parameters or send messages to start using

🔧 Function Description

📋 Home Page

  • Display project introduction and basic information
  • Related knowledge about AVHI (Acoustic Valley Hall Insulator)
  • View training data sources and method advantages

🎨 Design-AVHI Page

Dedicated parametric design for Acoustic Valley Hall Insulators:

  1. Enter target design parameters:
    • First Bandgap (Hz): Enter the target first topological bandgap frequency
    • Second Bandgap (Hz): Enter the target second topological bandgap frequency
    • ⚠️ Note: First bandgap must be less than second bandgap
  2. Click Run: System starts design and displays progress
  3. View results:
    • Design parameter comparison (target vs actual)
    • Lattice constant information
    • Structure images (PNG format)
    • Data files (TXT format)
  4. Download files: Click download buttons to save result files

💬 Chat Page

Conversational intelligent design and Q&A:

  1. Natural language interaction:

  3. Operation instructions:

❓ Frequently Asked Questions

Q: Why is there no response after entering parameters?

A: Please check:

  • Ensure both bandgap parameters are entered
  • Ensure first bandgap is less than second bandgap
  • Check if network connection is normal
  • Check browser console for error messages

Q: What to do if design fails?

A: Possible causes and solutions:

  • Parameters out of range: Try adjusting bandgap parameters to reasonable range
  • No match in database: System will prompt "Cannot find corresponding structure", please try other parameters
  • Network issues: Check API connection and network status

Q: How to download generated files?

A: After successful design, download buttons will be displayed in the results area:

  • Click "📥 Download Images" to download all structure images (PNG) at once
  • Click "📥 Download Data" to save structure data (TXT)
  • Click "🔧 Download STL" to save 3D printing file
  • Click "🔬 Comsol Model" to save simulation model file
  • Files will be automatically saved to the browser's default download directory

Q: How to restart conversation in Chat mode?

A: Click the "Start a new conversation" button to clear conversation history and restart the conversation.

⚡ Usage Tips

  • Parameter selection: Suggested bandgap range between 100-10000 Hz, intervals should not be too small
  • Chat interaction: Be as specific as possible when describing requirements, include clear numerical parameters
  • File management: Download and save design results promptly to avoid data loss
  • Multiple attempts: If one design is unsatisfactory, adjust parameters and redesign
  • Shortcut operations: Chat page supports Enter key for quick message sending
